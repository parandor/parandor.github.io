# parandor.github.io

One-page CV built with Vite + React + TypeScript and deployed to GitHub Pages.

## Local dev

```bash
npm install
npm run dev
```

## Edit CV content

Update `src/data/cvData.ts`.

## Deploy

1. In GitHub: **Settings → Pages → Source** = **GitHub Actions**
2. Push to `main` (workflow builds and deploys `dist/`)
